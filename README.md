# 19 - Chat en tiempo real

Descripción
- Chat en tiempo real usando Socket.IO y Express. Incluye servidor y cliente minimal.

Tecnologías
- Node.js, Express, Socket.IO, HTML/CSS

Cómo ejecutar
1. Abrir terminal en la carpeta del proyecto.
2. Ejecutar `npm install`.
3. Ejecutar `npm start`.
4. Abrir `http://localhost:3002` en varios navegadores/ventanas para probar.
